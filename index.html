Naming rules

  • Filenames: ####_Suffix.ext (e.g. 0001_BaseColor.png).
  • Suffixes (case-insensitive): High, BaseColor, Depth, Metallic, Normal, Reflection, Roughness.
  • High = Beauty/Base (always drawn first, no slider).
  • Other passes overlay the base with fade (sliders start at 0%).
  • Recommended: same resolution across all passes within a frame.
  • Multiple frames supported; use the timeline or type a frame number.
