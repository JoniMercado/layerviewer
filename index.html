Naming rules

  • Numeric-only files (e.g. 1.png, 025.jpg) become Combined 01, 02… (listed first).
  • Layer pattern (case-insensitive): BaseColor_#, Depth_#, Metallic_#, Normal_#, Reflection_#, Roughness_#.
  • High or High_# (if present) renders as base; if absent, layers still render fine.
  • Drag the ⋮⋮ handle to reorder layers (top renders first).
